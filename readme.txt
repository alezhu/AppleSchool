Тестовое задание для абитуриентов летней школы iPhone-разработчиков от e-Legion

Задача
Распарсить xml файл — отчет кассового аппарата и вывести сгруппированную информацию.

Язык разработки: C#.



Вся логика в Program.cs
Файлы для теста:
TestOrig.xml - первоначальный файл из задания
Test2.xml - файл, в который добавлены записи с невеным форматом цены, заказы с несовпадающими суммами по товарам, товары с пустыми типами

Логика не обрабатывает ситуации, когда в файле есть заказы с одинаковым ID, потому что в задании нет указаний на то как их обрабатывать (объединять, отбрасывать, выводить ошибку etc).

Сборка:
1 вариант. Открыть AppleSchool.sln в Visual Studio 2010 или в Visual C# 2010 Express и выполнить сборку (Build - BuildSolution)
2 вариант. Выполнить команду build.bat рядом с AppleSchool.bat

В результате в папке .\AppleSchool\bin\Debug\ появится файл AppleSchool.exe
Запускать: AppleSchool.exe <path_to_xml>.

Для сборки и работы требуется установленный .Net Framework 4.
